Local Delicacies API

Recipes

  • POST /recipes/: Create a recipe
  • GET /recipes/: Get all recipes
  • GET /recipes/{id}/: Get a particular recipe
  • PUT /recipes/{id}/: Update a recipe
  • DELETE /recipes/{id}/: Remove a recipe

Ingredients

  • GET /recipes/{id}/ingredients/: Get all ingredients for a particular recipe
  • GET /recipes/{id}/ingredients/{ingredientId}/: Get a particular ingredient of a recipe
  • POST /recipes/{id}/ingredients/{ingredientId}/: Create an ingredient for a recipe
  • PUT /recipes/{id}/ingredients/{ingredientId}/: Update the ingredient of a recipe

Tags

  • GET /recipes/tags/: Get all tags associated with recipes
  • POST /recipes/tags/: Create a tag
  • GET /recipes/tags/{id}/: Get a particular tag
  • PUT /recipes/tags/{id}/: Update a tag
  • DELETE /recipes/tags/{id}/: Remove a tag
